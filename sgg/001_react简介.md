# 001_react简介

## 是什么

React 

用于构建用户界面的JavaScript库

1. 发送请求获取数据
1. 处理数据(过滤,整理格式等)
1. **操作DOM呈现页面**

React是一个数据渲染为HTML视图的开源js库
